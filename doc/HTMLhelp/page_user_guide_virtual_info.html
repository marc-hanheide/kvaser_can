Virtual Channels

CANLIB supports virtual channels that you can use for development, test or demonstration purposes when you don't have any hardware installed.

To open a virtual channel,

  • call canOpenChannel() with the appropriate channel number, and
  • specify canOPEN_ACCEPT_VIRTUAL in the flags argument to canOpenChannel().

You can use canGetChannelData() and canGetNumberOfChannels() to list the currently available channels in the computer.
