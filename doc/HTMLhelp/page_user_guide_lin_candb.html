Accessing the Databases

The function calls in the Kvaser Database API all have names starting with kvaDb, e.g kvaDbOpen().

You will need to

#include kvaDbLib.h

in your programs, and link with kvaDbLib.dll (which, in turn, uses canlib32.dll).
